Na prática, como ela funciona?
Uma função pode estar associada a diversos usuários.
Cada usuário será associado a uma função.
Cada função contém um grupo de microatividades.
Função

Microatividade 2
Microatividade 3
  • Caso uma microatividade seja adicionada a uma função, todos os usuários que estão relacionados a ela receberão esse acesso.
  • As microatividades podem estar em mais de uma função.
Em caso de alteração na microatividade, todas as funções relacionadas herdarão as alterações realizadas.
