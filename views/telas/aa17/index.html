Estrutura de Acessos SAP
Função Organizacional
Atividade realizada pelo usuário dentro da organização. O perfil por função tem por objetivo criar um modelo para prover e administrar privilégios de acesso em uma organização. Neste modelo, cria-se o perfil de acesso baseado na função que o colaborador exerce dentro da organização.
Caixa de UND
Microatividade
Conjunto de transações agrupadas por atividade, por exemplo:
- microatividade 1: criar pedidos de compras;
- microatividade 2: exibir pedidos de compras.

de dinheiro
Transação
Código de acesso a telas para realizar determinada atividade no sistema SAP.
