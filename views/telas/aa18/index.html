Benefícios da Nova Estrutura de Acesso SAP
Clique nas garrafas para conhecer os principais benefícios da nova estrutura de acesso através do SAP:
Modelo alinhado com as melhores práticas para gerenciar os acessos de forma eficiente, de acordo com as funções, tarefas e atribuições dos usuários na organização.
Maior dinamismo e flexibilidade para concessão e revogação de acessos.
Menor esforço para manutenção de perfis de acessos: um ajuste em uma função é ativado automaticamente para todos os usuários atribuídos.
Definição de funções com mapeamento das respectivas atividades.
Nenhuma microatividade contém riscos de segregação de funções.
Maior facilidade e flexibilidade para manutenção das funções, pois muitas mudanças exigem apenas a atribuição de microatividades para a função.
Diminuição de acessos críticos em operações do dia a dia.
